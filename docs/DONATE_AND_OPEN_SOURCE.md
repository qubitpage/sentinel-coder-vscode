# Support Sentinel Coder One Studio

Sentinel Coder One Studio is open source and maintained by QubitPage Research.

If Sentinel helps your work and you want to keep the project frequently updated, any contribution is useful to sustain development, testing, documentation, provider integrations, Web compatibility, security hardening, and Marketplace maintenance.

## Why donations matter

Open-source AI tooling has real maintenance costs:

- Provider API testing.
- Azure/OpenAI/Groq/OpenRouter model experiments.
- Marketplace packaging and web compatibility work.
- Security reviews and secret scans.
- Documentation, guides, screenshots, and examples.
- Time spent responding to issues and feature requests.

Donations are optional. The project remains open source, and donations help keep it actively maintained.

## Contribute without donating

You can also help by:

- Opening high-quality bug reports.
- Improving docs.
- Adding provider setup notes.
- Testing VS Code Web behavior.
- Sharing safe Agentic Profile templates.
- Reviewing code for security and stability.
