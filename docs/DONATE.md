## Why donations matter

Any amount is useful. Donations help sustain:

- frequent provider/model catalog updates
- Marketplace releases
- VS Code Web compatibility maintenance
- security hardening
- tests and release verification
- documentation and tutorials
- open-source community support

The project is open source. Donations help keep it maintained and available to the community.
