# Marketplace Release Notes - 3.16.13

Sentinel Coder One Studio 3.16.13 focuses on enterprise readiness, security hardening, documentation, and public onboarding.

## Highlights

- Enterprise user guide.
- Provider/model setup guide.
- Security and hardening guide.
- Troubleshooting guide.
- Donation/open-source sustainability page.
- Pitch deck and whitepaper documentation.
- Cleaner Marketplace/GitHub onboarding links.
- Continued webview hardening and secret-scan discipline.

## Core capabilities

- Single-model full-capability mode.
- Opt-in Agentic Profiles.
- Live provider/model/context discovery.
- Categorized model selector with Agentic Modes, Most Used, and provider/free/paid groups.
- Free-only Agentic profile templates for testing and budget-safe experimentation.
- Azure/OpenAI/Anthropic/Groq/OpenRouter/Ollama and OpenAI-compatible providers.
- VS Code Web support with Remote Tool Bridge guidance.
- Studio workflows for media, documents, generated artifacts, and reports.
- Operation-aware native tool routing.
- Security firewall scans and package hygiene.
